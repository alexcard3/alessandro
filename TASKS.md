# Metanet task registry

Use this file as the canonical lightweight registry for project tasks that are
not already tracked in a dedicated issue system.

## Active

- [ ] Move scattered project tasks here, or link to the external system that owns them.
- [ ] Review the daily closed-task report and decide whether any completed work needs follow-up documentation, tests, or deployment/version checks.

## Closed

Add completed tasks here as `- [x] ...` entries when they become consolidated.
The daily review scans closed Markdown task-list items and reports them without
modifying project code.
